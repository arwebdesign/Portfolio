    HEADER AND CTA

    What I hope to Practice/learn:

    This header and CTA is very interesting. The background image changes on a slide effect to the left, and the menu bar moves up while scrolling and remains fixed at the top once it gets there.

    I'm pretty sure this will require some jQuery to create such effects, but I like the idea of fixed menus, I believe they are more powerful than a menu that disappears while scrolling.

    How this relates to what I have/will learn in wdd:

    I have already done some galleries that rotate as this one, but they all require some additional libraries, so I am not sure of the creation of such effects using just HTML and CSS. When it comes to the menu bar, I know it can be fixed through CSS, but I don't know if it is possible to make it scrollable and then fixed at the top.
